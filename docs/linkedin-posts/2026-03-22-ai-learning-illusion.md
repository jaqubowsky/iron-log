# AI Learning Illusion - Coach Setup

Date: 2026-03-22
Language: en

## Final version

I've been using AI to learn code for months. Turns out I wasn't learning much.

I watched a video by @Marcin Czarkowski about the illusion of learning and it got to me - you read AI-generated code, feel like you understand it, but your brain just skips over it. That was me and I didn't even notice until I tried to build something without AI and struggled way more than I expected.

I'm going through NestJS now and I set up Claude Code as a coach that refuses to write code for me. It explains stuff, reviews what I wrote, quizzes me. Nothing else.

It's annoying. You're stuck on something, the AI has the answer right there, and it won't give it to you. Each session starts with a quiz on last time's material. I code, it nudges me when I'm off. At the end I have to explain what I built and why. Then I make Anki flashcards - 5 min a day.

It tracks where I get stuck between sessions too so I actually go back to stuff instead of just moving on.

Two weeks in. I built two full NestJS modules from scratch - database, relations, error handling. And I can explain every line, not just make it compile.

Not quitting AI. The opposite - once I actually understand what I'm looking at, I'll get way more out of it than I ever did.

If you've tried something similar let me know how it went. I'm still figuring this out.

## Draft v1

I watched a video about how AI creates an illusion of learning. You read the code, you think you understand it, but your brain never actually does the work. Hit close to home.

So I changed how I use AI to learn. I'm picking up NestJS and backend concepts right now. Instead of letting AI write code for me, I set up Claude Code as a coach that only explains, reviews, and quizzes me. No code generation.

Each session starts with a quiz on what I learned last time — if I can't recall it, I go back. While I code, AI guides me without giving answers. At the end I have to explain what I built in my own words. Then I generate Anki flashcards and review them daily so the knowledge doesn't disappear after a week.

It also tracks my weak points between sessions. That's the part I didn't expect to matter so much. After a few sessions it picked up that I kept struggling with the same things and I could actually target them instead of moving on and hoping it clicks later.

Two weeks in and the difference is real. I caught myself reading a Prisma migration in raw SQL and actually understanding what it does. Before this I didn't even know CREATE TABLE syntax. I'm not faster, but I'm way more confident in what I know.

I'm not quitting AI. I'm building the foundation so that when I use it, I can actually verify what it gives me and make better decisions.

Anyone else tried learning this way? Curious how others balance AI speed vs actually retaining things.
