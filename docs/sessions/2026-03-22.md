# Sesja 2026-03-22

## Recall challenge

Pytanie: "Wytłumacz flow Prisma Exception Filter — od wyjątku w service do response dla klienta. Jakie kody obsługujesz i na jakie HTTP statusy mapujesz?" (3/5) — Ogólny flow opisał dobrze (filter globalny, łapie wyjątki, zwraca JSON). Kody Prisma nie pamiętał (i argumentował że nie powinien — fair point, zmieniono pytanie na typy błędów). Typy: duplikat → 409, nie istnieje → 404, FK violation → początkowo 409, po naprowadzeniu sam skorygował na 400. Rozumie mapowanie, jedno się pomylił ale sam poprawił.

## Co robił

WorkoutTemplatesModule — pełny CRUD. Schema Prisma z explicit M:N (tabela łącząca z polem `order`), migracje SQL (przeczytał i wytłumaczył), walidacja exercise ID przed create/update, transakcje ($transaction dla atomic update), cascade delete na relacji template → tabela łącząca. DTO z ValidateNested, PartialType, OmitType, ArrayNotEmpty. Przetestował HTTP callami.

## Samodzielność (1-5)

3 — Cały kod pisał sam. Architekturę wypracował w dyskusji sokratejskiej (explicit M:N, walidacja w service, cascade delete). Potrzebował naprowadzania na: różnicę implicit vs explicit M:N, czytanie migracji SQL (nie znał SQL-a), logikę filter/map (nie umie sam napisać tablicowej logiki), ON DELETE CASCADE vs RESTRICT (3x tłumaczone zanim zrozumiał), PartialType + OmitType w UpdateDTO. Sam znalazł buga przy update (DTO wymagało name), sam kwestionował architekturę ("czemu omijamy SQL?", "pytania nie powinny być hardkodowane").

## Co poszło źle

- **Metody tablicowe** — nie umie napisać filter/map/find/some samodzielnie. Przyznał że "logikę generuje AI". Krytyczna luka na rozmowę — 5 prób zanim napisał poprawny filter, i to z podpowiedzią
- **ON DELETE CASCADE** — zrozumiał dopiero po 3 wyjaśnieniach. Myślał że `onDelete: Cascade` na relacji = "usuwaj ten model" zamiast "usuwaj mnie gdy referencja zniknie"
- **SQL od zera** — nie znał podstaw SQL (CREATE TABLE, ALTER TABLE, CONSTRAINT). Nauczył się czytając migrację, ale to oznacza że fundamenty baz danych są bardzo słabe
- **`==` zamiast `===`** — drobne ale na rozmowie red flag
